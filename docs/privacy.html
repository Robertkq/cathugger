🔒 Privacy Policy

For Cathugger Discord Bot

Last updated: August 8, 2025

✅ Quick Summary

Cathugger respects your privacy completely. We don't collect, store, or share any personal data. Zero tracking, zero databases, zero worries.

📋 Our Privacy Commitment

The Cathugger Discord bot ("the Bot") is designed with privacy as a core principle. We do not collect, store, or share any personal data from users or servers.

🛡️ What We DON'T Do

  • No Data Storage: The Bot does not use any database or external storage systems.
  • No User Tracking: We don't track user behavior, preferences, or usage patterns.
  • No Message Logging: Bot commands and interactions are not logged or stored.
  • No Third-Party Sharing: No data is transmitted to external services or analytics platforms.
  • No Cookies or Tracking: This website and the bot don't use cookies or tracking mechanisms.

⚡ How the Bot Works

  1. Command Received: User runs a /cute command
  2. Processing: Bot processes the request in memory only
  3. Response: Bot sends a cute image from local files
  4. Memory Cleared: All command data is immediately discarded

The entire process happens in real-time without any persistent storage.

🔧 Technical Implementation

💾 Memory-Only Processing

All bot operations happen in RAM and are discarded when the process ends. No persistent storage exists.

📂 Local Files Only

Images are served from local files included with the bot. No external APIs or services are used.

🔍 Open Source

The entire codebase is public on GitHub. You can verify our privacy claims by reviewing the code.

🔮 Future Changes

If we ever add features that require data collection in the future:

  • This privacy policy will be updated with clear details
  • Users will be notified through the bot and website
  • All data collection will be opt-in and transparent
  • You will always have control over your data

Currently, there are no plans to add any data collection features.
