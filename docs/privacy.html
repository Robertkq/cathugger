Privacy Policy for Cathugger Discord Bot

The Cathugger Discord bot (“the Bot”) does not collect, store, or share any personal data.

Details:

  • The Bot does not use a database or external storage.
  • Any data processed (e.g., message content or user input) is handled only in memory during execution and discarded when the Bot stops running.
  • No personally identifiable information (PII) is stored or transmitted to third parties.

If data collection is added in the future, this policy will be updated accordingly.

If you have questions, please contact the developer via Discord or the repository page.
