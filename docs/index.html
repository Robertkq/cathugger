🚀 Quick Setup

  1. Clone the repository: git clone https://github.com/Robertkq/cathugger.git
  2. Install dependencies and build with CMake
  3. Get your Discord bot token from the Discord Developer Portal
  4. Run the bot: ./cathugger YOUR_BOT_TOKEN

Detailed setup instructions available in the repository README

🚀 How to Use

  1. Invite Cathugger to your Discord server using the button above
  2. Type /cute in any channel
  3. Choose either cat or dog
  4. Enjoy the cuteness! 🥰
